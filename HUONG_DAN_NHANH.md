# 🚀 HƯỚNG DẪN NHANH - 3 BƯỚC

## Bước 1: Cài Đặt (1 phút)

```bash
pip install -r requirements.txt
```

## Bước 2: Khởi Động (30 giây)

**Windows:**
```bash
START_WEB_APP.bat
```

**Linux/Mac:**
```bash
streamlit run traffic_web_app.py --server.port 5173
```

## Bước 3: Sử Dụng (2 phút)

1. Mở browser: `http://localhost:5173`
2. Upload video
3. Click **"🚀 Bắt Đầu Nhận Diện"**
4. Xem kết quả!

## 📊 Kết Quả

- **Video:** `output_video/YYYYMMDD_HHMMSS.mp4`
- **CSV:** `traffic_statistics.csv`
- **JSON:** `traffic_statistics_frames.json`
